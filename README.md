# How to run
1. Download the project in any directory
2. Go to that directory: `cd /path/to/project`
3. Run PHP: `php -S localhost:8000`
4. Open the app: http://localhost:8000/index.html

# Description
This is a single page application that allows you to search for artists on Spotify.com and display the result sorted alphabetically by the artistâ€™s name and popularity.
